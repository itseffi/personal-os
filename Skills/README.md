## How Skills Work

1. AGENTS.md references the relevant skill
2. Claude loads the skill's structured process
3. Follow the guided workflow
4. Get consistent, high-quality outputs

## Key Principles

1. **Evidence before claims** - Never claim success without verification
2. **One thing at a time** - Focus on single steps, not bundles
3. **Delete and restart** - When stuck, don't patch; start fresh
4. **Document as you go** - Capture learnings for future sessions
