# Opportunity Mapping Pipeline

Move from research evidence to a single target opportunity decision.

## When to Use

Use during continuous discovery and prioritization.

## Steps

1. Intake and normalize inputs
- Skill: `ost-intake`

2. Build opportunity tree
- Skill: `opportunity-solution-tree`

3. Select one target opportunity
- Skill: `ost-target-selection`

## Exit Criteria

- Selected opportunity is small, distinct, and moment-scoped.
- Recommendation is traceable to interview evidence.
