# Stakeholder Politics Copilot

Reduce political risk and improve adoption for feature and strategy proposals.

## When to Use

Use before circulating PRDs, updates, or strategy memos.

## Steps

1. Build baseline stakeholder map
- Skill: `power-map`

2. Review stakeholder risks for proposal
- Skill: `stakeholder-risk-review`

3. Create audience-specific framing
- Skill: `message-framing-comms`

4. Review executive update quality
- Skill: `executive-update-review`

5. Generate challenge questions for prep
- Skill: `challenging-stakeholder-questions`

## Exit Criteria

- Risks have mitigations and owners.
- Comms plan includes sequence, CTA, and measurable signals.
