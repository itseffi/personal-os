# Core Strategy Development

Develop strategy from context to execution with explicit tradeoffs.

## When to Use

Use when defining or revising product strategy.

## Steps

1. Diagnose strategic crux
- Skill: `crux-diagnosis`

2. Evaluate competitor posture
- Skill: `competitor-analysis`

3. Draft structured strategy
- Skill: `structured-product-strategy`

4. Model limit-based trajectory
- Skill: `limit-based-strategy`

5. Map value chain and vulnerabilities
- Skill: `value-chain-mapping`

## Exit Criteria

- Strategy includes objective, pillars, roadmap direction, and assumptions.
- Value chain vulnerabilities and defensive levers are explicit.
